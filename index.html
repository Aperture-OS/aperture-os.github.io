ApertureOS is a New Linux-based Distribution established in 2025 with the only goal of providing a wonderful, simple and intuitive user-friendly experience, making it the perfect choice for both newbies and experienced users. ApertureOS provides a set of tools like Blink, a powerful source-based package manager, and Pulse, its own innovative installer, capable of customizing your system in just a few clicks.

About ApertureOS

ApertureOS got established in 2025 by Elia73333 on the All Things Linux who wanted to create a Linux distribution that prioritizes user experience and simplicity. ApertureOS aims to provide a seamless and enjoyable computing experience for users of all skill levels.

Features

  • Performance: Fully optimized ISO file and filesystem with little to no bloat while still providing the right tools for a wonderful user experience.
  • Privacy: ApertureOS will NEVER collect any telemetry or user data.
  • Tools: ApertureOS provides tools such as Pulse, its own TUI/GUI installer that helps users customize their operating system with ease in every way possible.
  • Community: ApertureOS relies on community feedback to grow and fix its bugs, if you notice a bug, go open an Issue in our Github.
